### **班会**

> 我们每天的班会大家可以提任何问题，包括但不限前端的发展呀，程序员的生活呀，前端在公司的地位呀，程序员怎么保养自己的头发呀，以及如何学好前端这些，通过交流能够了解更多跟自己以后从事的工作相关的东西

- 时间：每天开（具体时间点投票决定）

- 形式：线上（飞书会议）

- 内容：当天课程内容相关知识提问；小组为单位提前准备好问题（前端，或者其他问题都可以）班会上问

- 回答问题：答疑讲师

**表现突出同学面试未通过可复活**
