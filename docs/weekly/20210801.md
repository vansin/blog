### docker使用技巧

docker 默认关闭挂载的功能，需要docker run时加上如下命令

--privileged=true 

```shell
docker run --gpus all --rm -ti --ipc=host --privileged=true -v $(pwd):/app fangshancheng/fastai:torch1.1 /bin/bash 
```
