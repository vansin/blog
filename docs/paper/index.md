# Paper专题


本专题主要聚焦在**前端智能化**领域，并涵盖其他相关领域的内容。

- 调研了解业界**前端智能化**落地基本情况，全方位了解其释放的生产力及不足。
- 复现**前端智能化**领域经典论文，并在此基础上改进从而达到研究生期间毕业要求。
- 掌握机器学习和深度学习领域基本方法论和经典模型的使用
