## 贡献及结果

本文发表在CVPR2020上

我们尽量专注于有效地使用少量数据来实现高精度结果。 朝着这个目标努力，我们的主要策略包括：

- 使用相对复杂但高效的 CNN 架构，在对象检测和分割基准数据集上获得高精度作为该方法的主要组成部分。
- 使用迭代迁移学习方法逐渐训练 CNN 模型，从更一般的任务开始，走向更具体的任务。 多次迭代迁移学习，从少量数据中有效提取所需知识。
